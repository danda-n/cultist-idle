# v0.8 Issues — Design Gaps After Full Review

Audit of CONCEPT.md v0.8. Every item here is something an implementer would have to guess,
a decision that hasn't been made, or a contradiction that will cause problems.

Organised by severity. **All items marked BLOCKS must be resolved before the affected system
is implemented.** Items marked DESIGN DECISION need a direction chosen. Items marked GAP
need the spec filled in.

> **v0.9 update:** All BLOCKS (B1–B6) and the MATH issue (M1) resolved. Remaining open:
> G1–G6 (design gaps) and D1–D6 (design decisions). See CONCEPT.md v0.9 for resolved specs.

## ~~BLOCKS IMPLEMENTATION~~ — ALL RESOLVED IN v0.9

### ~~B1 — Conjuring mechanic is undefined~~ ✅ RESOLVED
**Resolution (v0.9):** 8 Anima per conjure, 8s cooldown (fill bar). "Speed" = cooldown reduction.
Altar T1 → 6s, T2 → ~4.8s. Blood Compact (Phase 1 node 3) automates it. See §5.1.

---

### ~~B2 — Sacrifice mechanics are undefined~~ ✅ RESOLVED
**Resolution (v0.9):** +6 Anima/min per sacrifice, flat, stackable. Multiple sacrifices
intended. Ossuary → +9/min per sacrifice. Cindermark doubles all sacrifice production
retroactively. See §6.5.

---

### ~~B3 — Gateway construction cost is never stated~~ ✅ RESOLVED
**Resolution (v0.9):** 250 Anima. Buildable from M3 onward. Research node 2 (The Opened
Way) reduces cost by 20% on subsequent runs. See §8.4.

---

### ~~B4 — Expedition base timers are undefined~~ ✅ RESOLVED
**Resolution (v0.9):** Planet A: 20/15/12 min (1/2/3 cultists). Planet B: 35/26/20 min.
~25% reduction per extra cultist. See §11.1.

---

### ~~B5 — Research tree structure is underspecified~~ ✅ RESOLVED
**Resolution (v0.9):** Phase 1 = 4 named nodes (Conjuring Rites, The Opened Way, Blood
Compact, Heretical Wards), 180 Gnosis total. Phase 2 = 3 nodes per branch. Sustained
Channel is Phase 1 node 2. Overseer's Rite is Automation branch node 1. Full spec in §10.

---

### ~~B6 — Gateway capacity upgrade has no mechanism~~ ✅ RESOLVED
**Resolution (v0.9):** Per-gateway direct purchase in the gateway panel (Anima + Gnosis).
Not a construct or research node. Costs in §8.5.

### ~~G1 — Devotion "normal rate" is never stated~~ ✅ RESOLVED
**Resolution (v1.0):** Normal rate = 0.5%/min after M7. Player disciplines at ~70%,
roughly every 60 min base. Full decay table with modifier effects in §7.

### ~~G2 — Expedition outcome probability model is undefined~~ ✅ RESOLVED
**Resolution (v1.0):** `lost% = max(0, (50 − devotion%) × 0.6)`. Choice always 40%
independent of devotion. Cultist count affects speed only. Both planets same formula.
Probability table in §11.2.

### ~~G3 — Expedition Voltis sustain cost is undefined~~ ✅ RESOLVED
**Resolution (v1.0):** Planet A: 5 Voltis/min. Planet B: 8 Voltis/min. Per-minute flat
rate. 2 parallel expeditions = 2× drain. Spec in §11.1.

---

### ~~G4 — Corruption scope is contradictory~~ ✅ RESOLVED

defined (The Gnawing, Ash-Taint, The Shuddering Rite, Veil Sickness, Voltis Drain).
Cleansing cost: 200 Gnosis flat. Full spec in §11.4.

---

### ~~G5 — "Cultist automation unlock" research node effect is unclear~~ ✅ RESOLVED
**Resolution (v0.9):** It's Blood Compact (Phase 1 node 3) — automates conjuring,
fires on cooldown without clicking. Named and specced in §10.

## ~~DESIGN DECISIONS~~ — ALL RESOLVED

### ~~D1 — The Summoning (M14) has no spec~~ ✅ RESOLVED
**Resolution (v0.11):** 6th artifact auto-triggers the ritual — no separate button.
5-minute ritual phase: resources drain into portal, devotion collapse slows by 50%
(cannot fail), Harmony bonus shortens to ~3 min. M14 fires on completion. See §14 M14 note.

---

### ~~D2 — The ritual circle fill mechanic is undefined~~ ✅ RESOLVED
**Resolution (v0.11):** 6 segments, one per artifact. Crafted artifact segments fill
continuously with cost progress. Discovered artifact segments fill when expedition is
active. No separate mechanic — the circle IS the artifact system visualised. See §16.

---

### ~~D3 — Hungering Lens creates a dormant artifact~~ ✅ RESOLVED
**Resolution (v0.11):** Intentional. Found at M8 via Planet A, activates when Planet B
opens at M9. UI shows "Awaiting activation — requires Planet B gateway." Counts toward
M11 immediately on completion. See §12 Hungering Lens note.

---

### ~~D4 — Multiple sacrifices are never addressed~~ ✅ RESOLVED
**Resolution (v0.9):** Explicitly intended. Each sacrifice +6 Anima/min flat, stackable.
See §6.5.

---

### ~~D5 — Phase 2 research reset on Rehearsal is never stated~~ ✅ RESOLVED
**Resolution (v0.9):** Phase 2 resets each run. See §10.

---

### D6 — Expedition destinations: Planet A is one place or many?
**Affects:** #12 (Expeditions), §11, artifact sequencing

Before M9, all expeditions go to Planet A. But Voidwreath (M5) and Hungering Lens (M8)
both have expeditions that "go to Planet A." Do they:
- Compete for the same expedition slot (you can only run one "Planet A" expedition at a time)?
- Go to different sub-locations on Planet A?
- Both just pull from the same loot table with milestone-gated artifact eligibility?

Also: if you have 2 expedition slots and send both to Planet A simultaneously, can you
find Voidwreath and Hungering Lens on the same expedition pair, or is there one-at-a-time
artifact discovery?

### ~~M1 — Research Gnosis budget likely doesn't work~~ ✅ RESOLVED
**Resolution (v0.9):** Phase 1 fixed at 180 Gnosis (30+40+50+60). Phase 2 branches fixed
