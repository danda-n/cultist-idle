# v0.8 Issues — Design Gaps After Full Review

Audit of CONCEPT.md v0.8. Every item here is something an implementer would have to guess,
a decision that hasn't been made, or a contradiction that will cause problems.

Organised by severity. **All items marked BLOCKS must be resolved before the affected system
is implemented.** Items marked DESIGN DECISION need a direction chosen. Items marked GAP
need the spec filled in.

> **v0.9 update:** All BLOCKS (B1–B6) and the MATH issue (M1) resolved. Remaining open:
> G1–G6 (design gaps) and D1–D6 (design decisions). See CONCEPT.md v0.9 for resolved specs.

## DESIGN GAPS (spec incomplete)

### G1 — Devotion "normal rate" is never stated
**Affects:** #9 (Devotion), §7, balance

§7 defines the slow pre-M7 rate (~1.5%/10 min). M7 accelerates to "normal rate." The normal
rate is never stated.

Without it:
- Can't implement decay
- Can't verify "collapse requires sustained neglect"
- Can't confirm Discipline frequency matches design intent (needed every 60–90 min)

The normal rate also determines whether the Warding Stones construct (-20% decay) and the
"Devotion never drops below 30%" talent node are meaningful or trivial.

---

### G2 — Expedition outcome probability model is undefined
**Affects:** #12 (Expeditions), #25 (Success formula), §11.2

Outcome table shows Clean find / The Choice / Lost but gives no numbers.

- At what devotion level does "Lost" first become possible?
- Is it a hard threshold (below 30% = risk) or a continuous probability curve?
- What does 100% devotion guarantee vs. 50% devotion?
- "~40% of returns trigger The Choice" — is this independent of devotion, or does devotion
  affect Choice vs Clean find split as well?

Open question #2 defers this to #25. Fine, but the doc should at least name the variables
(devotion % at snapshot, destination, cultist count) so #12 and #25 can be built consistently.

---

### G3 — Expedition Voltis sustain cost is undefined
**Affects:** #12 (Expeditions), §11.1, Trifecta balance

§11.1: "each active expedition costs Voltis to sustain the open gateway."

- How much? Per minute? One-time on departure?
- Does running 2 parallel expeditions cost 2× the Voltis?
- Is this what makes parallel expeditions a meaningful tradeoff against Voltis reserves,
  or is it a trivial background cost?

---

### G4 — Corruption scope is contradictory
**Affects:** #21 (Corruption), §11.3, §11.4, §12

§11.4 says corruption "scales with later layers — parked for Layer 1 scope."
But §11.3 (The Choice pool) includes the Voidwreath major choice that explicitly produces
a corrupted artifact, and "risk a corrupted return" appears in a minor choice too.
The corruption debuff list and cleansing costs are never defined.

Decision needed: Is corruption in Layer 1 or not?
- If yes: spec the debuff list and cleansing costs. Remove the "parked" note.
- If no: remove the corruption-producing choices from the pool. They can't exist without the mechanic.

---

### G5 — "Cultist automation unlock" research node effect is unclear
**Affects:** #11 (Research), §10, §6.3

Phase 1 lists "cultist automation unlock" as a research node. But cultists already
auto-distribute via the priority system from game start (§6.3). What does this node do?

Candidates:
- Enables the passive Anima production from sacrifice (but that triggers at M3 via the
  sacrifice action itself, not research)
- Enables cultist auto-reassignment after stunts (but that's described as baseline behavior)
- Something else entirely

The name and effect need to be disambiguated before Phase 1 can be implemented.

---

### G6 — "Devotion upgrade tier" in Rehearsal persistence duplicates talent system
**Affects:** #15 (Prestige), §15, §13

§15 "What persists" lists: "Devotion upgrade tier." But §13.3 already lists 4 Devotion
talent nodes (higher starting devotion, faster recharge, 30% floor, keystone). These talents
persist via "Talent unlocks (permanent)" which is also in the same list.

"Devotion upgrade tier" is either:
- A duplicate of the Devotion talent nodes (remove it)
- A separate system not described anywhere (spec it)

## DESIGN DECISIONS NOT MADE

### D1 — The Summoning (M14) has no spec
**Affects:** #22 (The Summoning), §21 success criteria

M14 is described only as "Layer 1 complete." §21 says it must "feel like an event — not
a button press." That's an intent, not a design.

This is the game's entire reason for existing. Needs a decision on:
- Is it triggered automatically when the 6th artifact is placed, or does the player
  initiate a final ritual?
- Is there a final timer, sequence, or multi-step event?
- Any narrative text, visual sequence, or mechanical climax?
- Does anything in the player's current state affect how it plays out?

---

### D2 — The ritual circle fill mechanic is undefined
**Affects:** #16 (UI), §16, player motivation

§16: "Summoning progress — ritual circle, always filling."

If the circle fills only on artifact completion (6 fills in 10–12 hours), it's not "always
filling" — it's stationary for hours. If something else drives continuous fill, that
mechanism isn't defined.

Options to decide between:
- Artifact-only (6 steps, infrequent but dramatic)
- Milestone-triggered (14 steps across the run)
- Continuous slow fill based on Anima generated, punctuated by artifact jumps
- Some combination

This affects how present the goal feels during the long idle stretches.

---

### D3 — Hungering Lens creates a dormant artifact
**Affects:** #13 (Artifacts), §12, Rehearsal value

Hungering Lens expedition is available at M8. Its reward: "Voltis soft cap doubled,
automation costs -25%." Voltis doesn't exist until M9.

If completed at M8 and carried through Rehearsal, the artifact's bonuses are dormant
for the entire early game of the next run. Does it:
- Count toward M11 (first artifact) even while dormant?
- Show as "completed but inactive" in the artifact screen?
- Activate automatically when Planet B is discovered?

Also: is this intentionally the artifact you pick up before its effect matters, as a
forward investment? Or is this a sequencing mistake?

---

### D4 — Multiple sacrifices are never addressed
**Affects:** #8 (Sacrifice), §6.5, balance

With 5–8 starting cultists and a floor of 3, a player can sacrifice 2–5 cultists in a run.
The doc discusses sacrifice only in the singular. Needs a decision:

- Is sacrificing multiple cultists intended?
- If yes: does each sacrifice add the same flat passive Anima rate?
- If yes: stacking 4–5 sacrifices would compound significantly — is that intended power?
- Does Ossuary (+50% yield) and Cindermark (yields doubled) compound per-sacrifice?

---

### D5 — Phase 2 research reset on Rehearsal is never stated
**Affects:** #15 (Prestige), #11 (Research), §15

§15 says "All Phase 1 research nodes auto-complete on next run." Phase 2 is silent.

Presumably Phase 2 resets (player re-chooses path each run — good design). But it needs
to be explicit, since:
- Scrying Pool (+25% research speed) affects Phase 2 purchase speed on subsequent runs
- Whisperlock artifact (Research speed +50%, Phase 1 auto-complete) implies Phase 2
  doesn't auto-complete — if it did, Whisperlock's Phase 1 benefit would be redundant
- Players need to know before Rehearsing whether their Phase 2 investment carries over

---

### D6 — Expedition destinations: Planet A is one place or many?
**Affects:** #12 (Expeditions), §11, artifact sequencing

Before M9, all expeditions go to Planet A. But Voidwreath (M5) and Hungering Lens (M8)
both have expeditions that "go to Planet A." Do they:
- Compete for the same expedition slot (you can only run one "Planet A" expedition at a time)?
- Go to different sub-locations on Planet A?
- Both just pull from the same loot table with milestone-gated artifact eligibility?

Also: if you have 2 expedition slots and send both to Planet A simultaneously, can you
find Voidwreath and Hungering Lens on the same expedition pair, or is there one-at-a-time
artifact discovery?
